            Controllers 1 and 2 work but 1 has no programmed inputs

            UP = UP ARROW, DOWN = DOWN ARROW, LEFT = LEFT ARROW, RIGHT = RIGHT ARROW,

            START = ENTER, SELECT = SHIFT, A = X, B = Z, X = S, Y = A, L = D, R = C. 

            - Pressing L will toggle 'log-mode', where it logs CPU instructions.

            - Pressing P will toggle 'no-PPU mode', where it will not emulate the PPU rendering.
